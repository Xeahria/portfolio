Freeman's Shop

A Discord bot I developed for managing server-based interactions and enhancing user experience. The bot includes a welcome system for new members, a ticket system for support, a verification system to ensure proper member access, and features for monitoring member activity and tracking statistics for both members and customers in voice channels. It's built with Node.js and Discord.js.

TechCore Server Monitoring

This bot efficiently manages virtual servers and companies, monitoring all installed servers, reporting incidents, and allowing users to submit reports when issues arise. Built with Node.js and Discord.js, it is designed to meet the needs of virtual businesses with real-time updates and automated server management.
